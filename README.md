🍀 부자 되기 프로젝트 💸 (Lotto Number Generator)
1. 프로젝트 개요
"당신에게 행운이 깃들기를 바랍니다"
이 프로젝트는 단순한 난수 생성을 넘어, 사용자가 로또 번호를 쉽고 재미있게 관리하고 당첨 결과를 즉시 확인할 수 있도록 돕는 웹 기반 로또 관리 애플리케이션입니다. 모바일 환경에 최적화된 UI와 직관적인 기능을 제공합니다.
2. 주요 기능 (Key Features)
🎲 번호 생성 및 입력
 * 자동 생성: 1~45 사이의 중복 없는 숫자 6개를 무작위로 생성합니다.
 * 수동 입력: 사용자가 원하는 번호 6개를 직접 선택하여 저장할 수 있습니다.
 * 이미지/QR 스캔 (AI 기능):
   * QR 코드 인식: jsQR 라이브러리를 사용하여 로또 용지의 QR 코드를 인식, 번호를 추출합니다.
   * OCR (광학 문자 인식): Tesseract.js를 활용하여 이미지 내의 텍스트(숫자)를 분석하고 번호를 조합합니다.
   * 이미지 전처리: 인식률을 높이기 위해 이미지를 2배 확대하고 대비를 조절하는 전처리 로직이 포함되어 있습니다.
📝 기록 관리 (History)
 * 회차별 관리: 현재 회차뿐만 아니라 과거/미래 회차를 선택하여 기록을 분리해서 관리할 수 있습니다.
 * 로컬 저장소 활용: 브라우저의 localStorage를 사용하여 새로고침 하거나 창을 닫아도 데이터가 유지됩니다.
 * 삭제 기능:
   * 개별 삭제: 특정 게임만 선택하여 삭제할 수 있습니다.
   * 회차 전체 삭제: 현재 선택된 회차의 모든 기록을 한 번에 삭제할 수 있습니다. (실수 방지 모달 포함)
🏆 결과 확인 (Result Checking)
 * 실시간 당첨 확인: 동행복권 공식 API를 호출하여 해당 회차의 당첨 번호를 가져옵니다.
 * 자동 매칭 시스템: 저장된 나의 번호와 당첨 번호를 비교하여 등수(1등~5등, 낙첨)를 자동으로 판별합니다.
 * 시각적 피드백:
   * 당첨된 번호는 원래 색상 유지 및 강조 효과.
   * 낙첨된 번호는 회색(Dimmed) 처리하여 직관적으로 결과를 보여줍니다.
   * 보너스 번호 구분: 2등 당첨 시에만 보너스 번호를 보라색 테두리로 강조하여 1등과 혼동을 방지합니다.
 * 결과 요약 배너: 전체 게임 중 1등~5등이 각각 몇 건 당첨되었는지 요약 정보를 제공합니다.
3. 기술 스택 (Tech Stack)
 * Core: HTML5, JavaScript (ES6+)
 * Styling: Tailwind CSS (CDN) - 반응형 디자인 및 유틸리티 클래스 활용
 * Libraries:
   * jsQR: QR 코드 스캔 기능
   * tesseract.js: 이미지 OCR 분석 기능
   * FontAwesome: 직관적인 아이콘 사용
4. UI/UX 디자인 특징
 * 모바일 퍼스트 (Mobile First): 스마트폰에서 사용하기 편하도록 터치 영역을 최적화하고 세로 스크롤 레이아웃을 적용했습니다.
 * 현실적인 디자인: 실제 로또 공 색상(노랑, 파랑, 빨강, 회색, 초록)을 그대로 적용하여 익숙함을 줍니다.
 * 사용자 경험(UX) 고려:
   * 스크롤 최적화: 게임 기록이 많아질 경우 '무한 스크롤(Lazy Loading)' 방식을 적용하여 초기 로딩 속도를 확보했습니다. (기본 20개씩 로드)
   * 직관적인 인터랙션: 결과 확인 시 축하 메시지(꽃다발 이모지 등)와 애니메이션 효과를 제공합니다.
   * 자동 회차 계산: 오늘 날짜를 기준으로 가장 최신의 구매 가능한 회차를 자동으로 계산하여 보여줍니다.
